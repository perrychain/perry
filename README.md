## Mission

Perry - Messaging blockchain made for speed.

## Building

Build the app (Go 1.18+ recommended)

`make build`

## Installation

Generate a new wallet

`./bin/perry keygen new`

Launch an instance of the Perry blockchain

`./bin/perry serve`
